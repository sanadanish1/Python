# Python Programming Basics

## This repository covers the fundamentals of Python programming, including:

- Core data types: `int`, `float`, `str`, `list`, `tuple`, `set`, and `dict`
- Basic operations on different data types
- Introduction to **NumPy** for numerical computing
- Introduction to **Pandas** for data handling and analysis
- Introduction to **Matplotlib** for basic data visualization

The materials are suitable for beginners who want to build a strong foundation in Python for data analysis and scientific computing.
